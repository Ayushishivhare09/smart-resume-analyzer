## 🔥 Key Features

### 📄 Resume Analysis
- 📊 Get an instant **ATS score**
- 🔍 Keyword matching with job descriptions
- 💡 Identify **skill gaps**
- ✍️ Generate **AI-powered cover letters**

### 📊 Job Search Dashboard
- 📁 Manage multiple resumes
- 📌 Track applications (Applied → Interview → Offer)
- 📈 Monitor your improvement over time

## 💭 Problem It Solves

Most people apply blindly without knowing:
- Why their resume gets rejected  
- What keywords are missing  
- How to improve for specific roles

## 💡 Solution

Smart Resume Analyzer provides **real-time, actionable insights** to:
- Improve ATS compatibility  
- Align resumes with job descriptions  
- Track and optimize your job search strategy
