# 🚀 Smart Resume Analyzer

An intelligent **AI-powered Resume Analyzer** that evaluates resumes against job descriptions and provides insights like **ATS score, keyword matching, and skill analysis**.

This project helps job seekers understand how well their resume matches a job description and highlights areas for improvement.

---

## ✨ Features

* 📄 Upload Resume
* 🧠 Analyze resume against Job Description
* 📊 ATS Score estimation
* 🔍 Keyword Matching
* 💡 Skill Gap Suggestions
* 📈 Match Score Dashboard
* 🖥️ Clean and responsive UI

## ⚙️ How It Works

1. Upload your resume
2. Paste a job description
3. The analyzer extracts:

   * Skills
   * Keywords
   * Relevant experience
4. The system calculates a **match score and ATS compatibility**
